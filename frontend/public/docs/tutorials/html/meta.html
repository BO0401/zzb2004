HTML文档的<meta>元素

<meta>元素用于提供关于HTML文档的元数据 （metadata），这些信息通常不直接显示在页面上，但对浏览器、搜索引擎和其他服务非常重要。

元数据可以包括文档的字符集、作者、描述、关键词、视口设置等。

常见的<meta>元素

  • <meta charset="UTF-8">：指定文档的字符编码，通常使用UTF-8。
  • <meta name="viewport" content="width=device-width, initial-scale=1.0">：设置视口属性，确保页面在移动设备上正确显示。
  • <meta name="description" content="页面描述">：提供页面的简要描述，有助于搜索引擎优化（SEO）。
  • <meta name="keywords" content="关键词1, 关键词2">：定义页面的关键词，有助于搜索引擎索引。
  • <meta name="author" content="作者名">：指定文档的作者。
  • <meta http-equiv="refresh" content="30">：设置页面自动刷新，每30秒刷新一次。

如何使用<meta>

在<head>部分中，你可以添加多个<meta>元素来提供不同类型的元数据：

<meta charset="UTF-8">

确保使用适当的元数据来提高页面的可访问性和搜索引擎排名。
