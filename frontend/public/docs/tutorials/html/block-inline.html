块元素与行内元素（内联元素）教学

概念介绍

HTML元素根据其显示特性可以分为两大类：块级元素（Block Elements）和行内元素（Inline Elements）。

块级元素（Block Elements）

特点：

  • 独占一行，从新行开始显示
  • 可以设置宽度、高度、内边距和外边距
  • 宽度默认为父容器的100%
  • 可以包含其他块级元素和行内元素

常见的块级元素：

div - 通用块级容器

p - 段落

h1-h6 - 标题

  • ul/ol - 列表
  • li - 列表项
blockquote - 引用块
form - 表单
table - 表格

行内元素（Inline Elements）

特点：

  • 在同一行内显示，不会换行
  • 宽度和高度由内容决定，无法直接设置
  • 只能设置左右的内边距和外边距
  • 只能包含其他行内元素和文本

行内块元素（Inline-Block Elements）

特点：

  • 结合了块级元素和行内元素的特点
  • 在同一行内显示，但可以设置宽度和高度
  • 可以设置所有的内边距和外边距

示例：

inline-block 1
inline-block 2
inline-block 3

CSS display 属性

可以通过CSS的 display 属性来改变元素的显示类型：

  • display: block; - 转换为块级元素
  • display: inline; - 转换为行内元素
  • display: inline-block; - 转换为行内块元素
  • display: none; - 隐藏元素
