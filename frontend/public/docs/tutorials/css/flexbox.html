Flexbox 弹性布局完全指南

现代CSS布局的核心技术，掌握轴向、对齐、空间分配与实战技巧。

1. 基本概念

容器（Container）：设置 display:flex 的元素。

项目（Items）：容器的直接子元素。

主轴（Main Axis）：主要方向，默认水平。

交叉轴（Cross Axis）：垂直于主轴的方向。

/* 基本设置 */ .container{ display: flex; }

2. 容器属性

flex-direction
/* 主轴方向 */ row | row-reverse | column | column-reverse
flex-wrap
/* 换行控制 */ nowrap | wrap | wrap-reverse
justify-content
/* 主轴对齐 */ flex-start | center | flex-end space-between | space-around | space-evenly
align-items
/* 交叉轴对齐 */ stretch | flex-start | center flex-end | baseline

3. 项目属性

flex-grow

定义放大比例，默认0（不放大）。

.item{ flex-grow: 1; }
flex-shrink

定义缩小比例，默认1（允许缩小）。

.item{ flex-shrink: 0; }
flex-basis

分配剩余空间前的初始大小。

.item{ flex-basis: 200px; }
flex 简写
/* flex: grow shrink basis */ flex: 1; /* 1 1 0% */ flex: auto; /* 1 1 auto */ flex: none; /* 0 0 auto */

5. 常见问题与解决

  • 最小宽度问题：项目默认不会收缩小于内容宽度，使用 min-width:0 或 overflow:hidden 解决。
  • 高度对齐：align-items:stretch 可让项目填满容器高度。
  • 换行布局：结合 flex-wrap:wrap 与 align-content 控制多行对齐。
