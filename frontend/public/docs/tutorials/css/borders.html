CSS 边框与圆角 borders 完全指南

掌握 border 宽度/样式/颜色、边框图片、圆角与椭圆角、轮廓 outline、阴影的对比。

1. 基础属性

  • border-width / border-style / border-color，或使用简写 border。
  • 单侧边框：border-top/right/bottom/left-*
  • 圆角：border-radius（支持每个角、椭圆角）
  • 边框图片：border-image
  • 轮廓：outline（不占空间，不会影响布局）

2. 示例

不同样式

椭圆圆角

.pill{ border-radius: 999px / 24px; }

边框图片

.img-border{ border:12px solid transparent; border-image: linear-gradient(45deg,#22d3ee,#a78bfa) 1; }

3. 实战建议

  • 优先使用 border-radius 构建卡片与徽章，配合阴影获得层次感。
  • outline 用于可访问性焦点高亮，不要滥用取消它。
  • 边框图片在高分屏上表现良好，注意与背景的对比。

4. 练习

  1. 实现一个胶囊按钮（椭圆角 + 渐变边框）。
  2. 制作卡片 hover 时边框色平滑过渡（结合 transition）。
  3. 给输入框提供可访问性的键盘焦点 outline 效果。
