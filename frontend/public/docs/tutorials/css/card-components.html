4. 设计要点与实践

  • 图片使用 overflow + 缩放，避免布局抖动。
  • 交互使用过渡与阴影提升，限定在 compositing 层。
  • 通过变量统一圆角、间距、边框等视觉基线。
  • 移动端优先，使用 auto-fit + minmax 构建自适应网格。
