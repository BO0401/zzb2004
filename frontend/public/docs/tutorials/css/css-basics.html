CSS 基础语法与使用方式 入门

目标：理解 CSS 是什么、如何引入、基本语法、选择器与优先级概念。

1. CSS 是什么

CSS（Cascading Style Sheets，层叠样式表）用于控制 HTML 文档的表现（颜色、字体、布局等）。“层叠”表示当多个规则作用于同一元素时，有一套优先级规则决定最终样式。

2. 在 HTML 中使用 CSS 的三种方式

方式 A：内联样式（Inline）

直接在元素上写 style 属性。

<h1 style="color:red;">标题</h1>

优点：简单；缺点：分散、难维护，不建议在项目中大量使用。

方式 B：内部样式（Internal）

在页面的 <head> 中使用 <style> 标签。

<style> h1{ color: red; } </style>

适合小页面或教学示例，本教程大量采用这种方式便于展示。

方式 C：外部样式（External）

将样式写入 .css 文件，通过 <link> 引入。

<link rel="stylesheet" href="styles.css">

企业开发推荐：结构与样式分离，利于复用与缓存。

3. 基本语法结构

选择器 { 属性: 值; 属性: 值; }

每条声明以分号结尾；多个声明用花括号包裹。属性名与值之间以冒号分隔。

4. 常见选择器

元素选择器
p { color:#374151; }
类选择器
.highlight { background: #fef3c7; }
ID 选择器
#main { max-width: 960px; }
组合选择器
article p { line-height:1.8; } /* 后代 */ article > p { } /* 子代 */ article + aside { } /* 相邻兄弟 */ article ~ p { } /* 通用兄弟 */

5. 优先级（Specificity）与层叠

优先级从低到高：元素 < 类 < ID < 内联样式；若优先级相同，后出现的规则覆盖先出现的（层叠）。

/* 同一元素多条规则：后者覆盖前者 */ p{ color: green; } p{ color: blue; }

避免滥用 !important，它会提升优先级，影响可维护性。正确做法是优化选择器与结构。

6. 基本调试与开发者工具

打开浏览器开发者工具（F12），在 Elements/样式面板实时修改CSS；在 Computed 面板查看最终生效的样式与继承关系。

7. 练习

  1. 为页面标题设置不同颜色和字号，并尝试用类选择器批量控制。
  2. 创建一个 .button 类，设置背景、内边距、圆角，并为不同按钮复用。
  3. 尝试写出三种引入方式，并比较它们的维护体验。
