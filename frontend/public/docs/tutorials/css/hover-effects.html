纯 CSS 悬停特效集（Hover Effects）

使用 transitions/transforms/shadows/gradients/伪元素等，构建无需 JS 的高质感悬停反馈。

1. 下划线滑入

Hover 我
.link{ background:linear-gradient(currentColor,currentColor) left bottom/0 2px no-repeat; }
.link:hover{ background-size:100% 2px; }

按钮高光划过

Shimmer
.btn::after{ background:linear-gradient(120deg,transparent,rgba(255,255,255,.5),transparent); }

2. 图片放大 + 变暗

.figure:hover img{ transform:scale(1.06) }

卡片悬浮阴影

卡片
.card:hover{ transform:translateY(-6px); box-shadow:0 16px 36px rgba(2,6,23,.55); }

3. 边框显现与 3D 倾斜

悬停显现边框
.reveal::before{ padding:2px; background:linear-gradient(135deg,#22d3ee,#a78bfa); mask-composite:exclude; }
3D Tilt
.tilt:hover{ transform:perspective(600px) rotateX(6deg) rotateY(-6deg); }

4. 练习

  1. 为导航链接设计“下划线从中间向两侧扩展”的效果。
  2. 为卡片添加“边框显现 + 阴影提升 + 细微放大”的组合交互。
  3. 为图片网格添加“放大 + 变暗 + 文本淡入”的悬停效果。
