CSS 过渡 transitions 完全指南

掌握 transition-property/duration/timing-function/delay 与简写，理解哪些属性可动画化，配合变换与颜色制作顺滑交互。

1. 基本语法

/* 完整写法 */
selector{ transition-property: transform, background-color; transition-duration:.3s; transition-timing-function:ease; transition-delay:0s; }
/* 简写 */
selector{ transition: transform .3s ease 0s, background-color .3s; }

常见缓动函数：ease、linear、ease-in、ease-out、ease-in-out、cubic-bezier(x1,y1,x2,y2)。

4. 实战建议

  • 优先对 transform/opacity 做过渡，性能更佳。
  • 动画时间 150–300ms 更易被接受，具体视场景而定。
  • 避免为所有属性写 transition: all，可能导致意外过渡。

5. 练习
