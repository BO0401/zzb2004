CSS 溢出 overflow 完全指南

理解 overflow/overflow-x/overflow-y、滚动轴行为、滚动截断、滚动条与触发 BFC 的作用。

1. 常用取值

  • visible（默认）：内容可能溢出容器边界。
  • hidden：裁剪超出部分，不出现滚动条。
  • scroll：强制显示滚动条（即便不溢出）。
  • auto：仅在需要时显示滚动条。
  • clip：类似 hidden，但不创建滚动容器。

3. 轴向控制与滚动截断

.x-only{ overflow-x:auto; overflow-y:hidden; }
.y-only{ overflow-y:auto; overflow-x:hidden; }

文本单行截断常用 white-space: nowrap; overflow: hidden; text-overflow: ellipsis;。

4. BFC 与自适应高度

为父容器设置 overflow:auto/hidden 可触发 BFC，常用于包含浮动或避免外边距折叠。
