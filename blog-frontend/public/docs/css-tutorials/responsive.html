响应式设计

适配不同设备屏幕，创建跨平台一致的用户体验。

1. 基础概念

响应式设计通过媒体查询（Media Queries）、弹性布局、可缩放图片等技术，让网页在不同屏幕尺寸下都有良好体验。

viewport meta 标签是响应式的前提：

<meta name="viewport" content="width=device-width, initial-scale=1.0" />

2. 媒体查询语法

/* 基本语法 */ @media 媒体类型 and (条件) { /* 样式规则 */ } /* 常用示例 */ @media screen and (max-width: 768px) { .mobile-only { display: block; } } @media (min-width:769px) and (max-width:1024px) { .tablet-layout { grid-template-columns: 1fr 1fr; } }

3. 断点策略

常见断点：
  • 小于 480px：超小屏（手机）
  • 480px - 768px：小屏（大手机/小平板）
  • 768px - 1024px：中屏（平板）
  • 1024px+：大屏（桌面）

4. 移动优先 vs 桌面优先

移动优先：从小屏开始设计，逐步增强。

/* 移动优先 */ .container { width: 100%; } @media (min-width: 768px) { .container { width: 750px; } }

桌面优先：从大屏开始，逐步简化。

/* 桌面优先 */ .container { width: 1200px; } @media (max-width: 767px) { .container { width: 100%; } }

6. 图片响应式

/* 响应式图片 */ img { max-width: 100%; height: auto; } /* 或使用 picture 元素 */ <picture> <source media="(max-width:768px)" srcset="mobile.jpg"> <img src="desktop.jpg" alt="..."> </picture>
