CSS 变换 transforms 完全指南

掌握 translate/scale/rotate/skew、transform-origin、3D 变换与合成层优化。

1. 基本用法

.box{ transform: translate(10px, 0) scale(1.05) rotate(3deg); transform-origin: center; will-change: transform; }

3. 实战建议

  • 合成层优化：必要时使用 will-change: transform 或 transform: translateZ(0)，避免滥用。
  • 与 transition/animation 组合获得高性能动效。
  • 注意变换不影响文档流尺寸，必要时结合 layout 属性调整。
