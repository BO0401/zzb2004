3. 设计要点

  • 模态层需使用半透明遮罩并禁用底层交互；注意焦点管理（纯 CSS 方案受限）。
  • Toast 支持堆叠、自动消失与可操作按钮；使用阴影与圆角提升可读性。
  • 在真实产品中，建议配合 JS 管理可访问性的焦点与键盘关闭行为。
