CSS 背景 backgrounds 全面指南

掌握 background-color/image/position/size/repeat/attachment/origin/clip 与多重背景、渐变。

1. 基础属性

  • background-color 背景色
  • background-image 背景图或渐变
  • background-repeat 是否平铺
  • background-position 位置
  • background-size 尺寸（contain/cover/具体值）
  • background-attachment 滚动行为（scroll/fixed/local）
  • background-origin/clip 绘制区域

2. 示例

渐变背景

.box{ background: linear-gradient(135deg,#22d3ee,#a78bfa); }

cover 与居中

.hero{ background: url(hero.jpg) center/cover no-repeat; }

多重背景

.card{ background: radial-gradient(...), radial-gradient(...), #0b1220; }

3. 实战建议

  • 优先使用 SVG/渐变实现装饰，减少位图体积。
  • 大图使用 cover，注意焦点区域可配合 object-position（在 img/video）。
  • 使用多重背景实现光斑/纹理等效果，减少额外元素。

4. 练习

  1. 制作一个具有光斑渐变的卡片背景。
  2. 实现一个固定视差背景区块（background-attachment: fixed）。
  3. 实现一个图标平铺的无缝纹理背景。
