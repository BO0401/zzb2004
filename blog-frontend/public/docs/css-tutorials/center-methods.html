CSS 居中方案大全

涵盖 inline/line-height、margin auto、absolute+transform、Flexbox、Grid、未知尺寸、多行文本与图文混排等场景。

1. 水平居中

块级元素：margin:auto

box
.container{ width:600px; margin: 0 auto; }

内联内容：text-align:center

box 
.container{ text-align:center; }

2. 垂直居中

单行文本：line-height 等于高度

单行文本 
.title{ height:40px; line-height:40px; }

未知高度：absolute + transform

box
.box{ position:absolute; top:50%; left:50%; transform: translate(-50%,-50%); }

3. 双轴居中

Flexbox

box
.center{ display:flex; align-items:center; justify-content:center; }

Grid

box
.center{ display:grid; place-items:center; }

4. 多行文本与图文混排

多行文本垂直居中

多行文本\n多行文本\n垂直居中

.container{ display:flex; } .container p{ margin:auto; }

图文对齐

标题

副标题与图标垂直对齐

.media{ display:flex; align-items:center; gap:12px; }

5. 实战建议

  • 优先使用 Flex/Grid 的居中能力；未知尺寸时用 absolute+transform。
  • 表单内联元素对齐可使用 vertical-align: middle 或 align-items。
  • 复杂布局可抽象为“媒体对象模式”（media object）。

  2. 实现一个卡片列表，要求：图片与文本在垂直方向对齐，标题与按钮在两端对齐。
