1. 基本语法

@keyframes fade-in { from { opacity: 0; } to { opacity: 1; } }
.box{ animation-name: fade-in; animation-duration:.6s; animation-timing-function:ease; animation-delay:0s; animation-iteration-count:1; animation-direction:normal; animation-fill-mode:both; animation-play-state:running; }
/* 简写 */
.box{ animation: fade-in .6s ease both; }

3. 实战建议

  • 尽量动画化 transform 和 opacity，减少布局/绘制开销。
  • 为 prefers-reduced-motion 用户提供降级：尽量缩短或禁用动画。
  • 避免长时间、强对比、全屏范围的动画，以免造成眩晕。
@media (prefers-reduced-motion: reduce){ *{ animation-duration:.01ms !important; animation-iteration-count:1 !important; transition-duration:.01ms !important; } }
