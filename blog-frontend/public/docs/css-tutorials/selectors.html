CSS 选择器完全指南 核心

掌握选择器是写好 CSS 的基础。本页覆盖日常开发最常见的选择器及其优先级影响。

1. 基础选择器

元素（类型）选择器
p { color:#374151; }
类选择器
.btn { padding:8px 12px; }
ID 选择器
#app { max-width: 960px; }
通配选择器
* { box-sizing: border-box; }
优先级：ID > 类 > 元素；通配选择器不应频繁使用，注意性能与覆盖范围。

2. 组合器

后代
article p { }
子代
article > p { }
相邻兄弟
h2 + p { }
通用兄弟
h2 ~ p { }

3. 属性选择器

[attr]
[disabled] { opacity:.6; }
[attr="val"]
input[type="email"]{ }
[attr^="val"]
a[href^="https"]{ }
[attr$="val"]
img[src$=".svg"]{ }
[attr*="val"]
[class*="btn-"]{ }

4. 伪类（Pseudo-classes）

:hover :focus :active :visited :checked :disabled :first-child :last-child :nth-child(n) :not(selector)

5. 伪元素（Pseudo-elements）

带有视觉装饰的标题

伪元素可以在不改变HTML结构的前提下，添加装饰与内容。

6. 实战建议与常见误区

  • 尽量使用类选择器进行样式控制，避免依赖过深的层级选择器。
  • 不要将ID用作样式钩子，以免增加优先级导致维护困难。
  • 为组件设计语义化、可复用的类名（可结合 BEM 方法）。
  • 谨慎使用通配选择器与属性包含选择器，避免性能隐患。

7. 练习

  1. 为卡片列表编写选择器，实现奇偶行不同背景。
  2. 使用属性选择器为外部链接添加小图标（仅使用 ::after）。
  3. 写一个导航条，使用 :hover 与 :focus-visible 提升可访问性。
