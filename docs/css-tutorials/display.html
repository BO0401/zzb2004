display 属性全解析

控制元素的外部显示类型：是否换行、是否可设置宽高、是否参与文档流等。

1. 常见值

block（块级）
块级元素会独占一行
可以设置宽高与上下外边距
inline（行内）
行内1行内2行内3

行内元素不换行、不接受宽高，上下 margin 无效。

inline-block（行内块）
可设置宽高且不独占一行

2. none 与 visibility

  • display:none：元素不占据空间，完全从文档流中移除。
  • visibility:hidden：元素不可见，但仍占据空间。

3. flow-root 与格式化上下文

display:flow-root 可建立独立的 BFC（块级格式化上下文），常用于包裹浮动元素或避免外边距折叠。

4. 表格与列表显示

表格相关：table、table-row、table-cell；列表：list-item。通常不建议手写这些 display 值去模拟元素，语义优先。

5. 练习

  1. 将一组标签使用 inline-block 做成多行换行的标签云。
  2. 使用 flow-root 包裹浮动元素，观察高度塌陷是否被修复。
