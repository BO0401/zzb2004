CSS 颜色系统详解

掌握不同颜色表示方式与透明度应用，建立可维护的色彩体系。

1. 常见表示法

十六进制
color: #10b981; /* #RGB #RRGGBB #RRGGBBAA */
RGB / RGBA
color: rgb(59, 130, 246); color: rgba(59, 130, 246, 0.6);
HSL / HSLA
color: hsl(262 83% 58% / 1); color: hsl(262deg, 83%, 58%, .7);

HSL 更接近人类对颜色的理解，利于调整饱和度/明度。

2. 透明度

可通过 rgba/hsla 的 alpha 通道或使用 opacity 设置。注意：opacity 会影响子元素透明度；而颜色的 alpha 只作用于该属性。

rgba 半透明
opacity 半透明（含子元素）

3. 渐变（简单介绍）

background: linear-gradient(90deg, #60a5fa, #34d399);
background: radial-gradient(circle, #f472b6, #f59e0b);

4. 实战策略

  • 为项目建立色板：基础色、语义色（成功、警告、错误）、灰度。
  • 使用 CSS 变量存储颜色，提高一致性与可维护性。
  • 注意对比度（可访问性），确保文字与背景对比足够。
:root{ --color-primary:#3b82f6; --color-success:#10b981; --color-danger:#ef4444; }

5. 练习

  1. 用 HSL 构建一套按钮的颜色层级（hover/active）。
  2. 定义一组 CSS 变量管理项目色板，并在组件中使用。
  3. 将线性渐变用作卡片标题的底边高亮。
