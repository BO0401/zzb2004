CSS 单位详解

掌握 px、em、rem、%、vw、vh、vmin、vmax、ch 等单位的含义与使用场景。

1. 绝对单位 vs 相对单位

绝对单位

px（像素）在屏幕上是最常用的单位，通常作为基准。pt、cm、mm 在屏幕场景较少使用。

相对单位

em、rem 相对于字体大小；% 相对于父元素相关属性；vw/vh 相对于视口。

2. 字体相关单位：em 与 rem

em：相对于元素自身（或最近祖先）字体大小。嵌套可能导致累积放大。

.parent{ font-size:16px; } .child{ font-size:1.5em; } /* 24px */

rem：相对于根元素 html 的字体大小，常用于建立全局比例体系。

html{ font-size:16px; } .title{ font-size:2rem; } /* 32px */

3. 视口单位：vw / vh / vmin / vmax

vw：1vw 等于视口宽度的 1%。

vh：1vh 等于视口高度的 1%。

vmin/vmax：取宽高中较小/较大的那个维度。

.hero{ height:50vh; font-size:5vw; }

4. 宽度与百分比

% 对不同属性的参照不同：width 相对包含块的宽度，padding 相对自身宽度，line-height 的 % 相对元素字体大小。

.col{ width:33.33%; padding:2%; }

5. 其他有用单位

  • ch：大约等于“0”字符的宽度，适合限制文本宽度（如 60ch）。
  • ex：相对当前字体的 x 高度（较少用）。
  • clamp(min, preferred, max) 常与视口单位结合做响应式字号。
.title{ font-size:clamp(18px, 4vw, 36px); }

6. 练习

  1. 用 rem 建立 8pt 间距体系（如 0.5rem、1rem、1.5rem、2rem）。
  2. 创建一个响应式 hero 模块，使用 vh 设置高度，标题使用 clamp。
  3. 对比 em 与 rem 在嵌套组件中的差异。
