CSS 阴影与发光 shadows 完全指南

掌握 box-shadow / text-shadow、多重阴影、内阴影、柔光与霓虹发光效果。

1. box-shadow 语法

box-shadow: offset-x offset-y blur-radius spread-radius color inset;

可逗号分隔实现多重阴影；inset 表示内阴影。

4. 实战建议

  • 阴影颜色建议使用带透明度的深色，避免“脏”。
  • 移动端应控制阴影模糊半径与层数，减少重绘成本。
  • 发光效果可结合背景渐变/滤镜获得更自然的视觉。

5. 练习

  1. 制作一个悬浮卡片的悬停过渡阴影。
  2. 实现按钮的霓虹发光边框效果（内外发光叠加）。
  3. 给标题添加柔和 text-shadow，让字重更清晰。
