4. 实战建议

  • 组件内部优先使用 Flex；页面级或二维布局优先使用 Grid。
  • auto-fit/auto-fill + minmax 构建自适应卡片网格。
  • 使用 grid-template-areas 提升可读性与维护性。

5. 练习

  1. 实现「头/主/侧/脚」四区布局，侧栏固定 240px，其余自适应。
  2. 构建自适应卡片网格，最小列宽 200px。
  3. 使用区域语法重排布局顺序以适配窄屏。
