CSS 伪类与伪元素

掌握交互伪类（:hover/:focus/:active）、结构伪类（:first-child/:nth-child 等）与伪元素（::before/::after/::first-line/::first-letter）。

1. 交互伪类

按钮 
a:hover{ background:#132033; }
a:active{ transform: translateY(1px); }
a:focus{ outline:2px solid #22d3ee; outline-offset:2px; }

2. 结构伪类

  • 项目 1
  • 项目 2
  • 项目 3
  • 项目 4
li:first-child{ color:#93c5fd }
li:last-child{ color:#34d399 }
li:nth-child(odd){ background:#0a1327 }

3. 伪元素

使用 ::before/::after 注入装饰内容或结构辅助。

悬停我
.tooltip::after{ content: attr(data-tip); position:absolute; opacity:0; }
.tooltip:hover::after{ opacity:1; }

4. 练习

  1. 制作一个仅用 CSS 的 tooltip（如本页示例）。
  2. 使用 nth-child 实现斑马条表格。
  3. 为段落首字添加装饰性的首字下沉效果（::first-letter）。
